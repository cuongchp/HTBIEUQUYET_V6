# HỆ THỐNG BIỂU QUYẾT ĐIỆN TỬ - EVNCHP

## 📋 Mô tả dự án
Hệ thống biểu quyết điện tử cho EVNCHP với 9 module chính:
- 🏠 Trang chủ - Dashboard thống kê
- 📄 Dự thảo tờ trình 
- 🗳️ Biểu quyết
- 📊 Phiếu tổng hợp
- 📋 Quản lý nghị quyết
- 📁 Tủ tài liệu
- ✍️ Ký số văn bản
- 👥 Quản trị người dùng
- 🗑️ Thùng rác

## 🛠️ Công nghệ sử dụng
- **Backend**: Node.js, Express.js
- **Database**: SQL Server
- **Frontend**: Vanilla JavaScript, Bootstrap 5
- **Architecture**: SPA (Single Page Application)

## 📦 Cài đặt

### Yêu cầu hệ thống:
- Node.js >= 14.x
- SQL Server 2019+
- npm hoặc yarn
